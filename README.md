# Microfrontends with Module federation (Webpack 5+)

- Run both apps, host and remote.

Host in one terminal:

- `cd Host && npm run dev`

And in another terminal:

- `cd Remote && npm run dev`

Navigate to Home App: http://localhost:3009.

## Useful commands

- `npm run dev`
- `npm run build`
- `npm run server`
